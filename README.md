# wzh Home Assistant Add-ons

This repository contains Home Assistant add-ons developed by wzh.

## Add-ons

### v2raya

A modern web GUI client of Project V which supports V2Ray, Xray, SS, SSR, Trojan, NaiveProxy, HTTP(S) and SSH.

- **Version**: 1.0.0
- **Description**: A modern web GUI client of Project V
- **URL**: https://github.com/mzz2017/v2raya

## Installation

1. In Home Assistant, go to **Supervisor** > **Add-on Store**.
2. Click the three dots in the top right corner and select **Repositories**.
3. Add the URL of this repository: `https://github.com/yourusername/wzh_ha_addons`
4. Click **Add** to add the repository.
5. Refresh the page, and you should see the add-ons from this repository.
6. Click on the add-on you want to install and follow the instructions.

## Usage

After installation, you can access the v2raya web interface at `http://your-home-assistant-ip:2017`.
